# Auth Server

Authentication server for unboxd-cloud services.

## Overview

This is the authentication server for the unboxd-cloud platform, providing secure authentication and authorization services.

## Features

- User authentication
- Token management
- Session handling
- Security middleware

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
npm install
```

### Configuration

Configure the server via environment variables or `.env` file.

### Running

```bash
# Development
npm run dev

# Production
npm start
```
